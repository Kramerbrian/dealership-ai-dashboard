# DealershipAI – Canon Export (Thread Bundle)

This bundle captures the canonical decisions and UI/UX specs discussed:

## Included
- Canonized sound bite for landing + Relevance Overlay explainer
- Core Web Vitals (plain-English) block pattern (lock across app)
- Relevance Overlay spec (Visibility × Proximity × Authority × Schema Weight)
- Schema Confidence Score (SCS) formula + competitor ranking
- Marketplace Generative Citations: structure, lists, filters, suppression
- Hover cards library (Factor → Why it matters)
- Grid + Modal UX for Crawl Errors + CWV diagnostics
- Component stubs for rapid implementation

## How to use
1. Drop these files into your Next.js repo.
2. Wire component stubs to your data sources (ATI/CIS/Pulse/Schema probe).
3. Use `data/canon/*` and `data/marketplaces.json` to seed initial tiles.
4. Replace any placeholder data models with your real APIs.
