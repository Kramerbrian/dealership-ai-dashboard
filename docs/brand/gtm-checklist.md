# Clay-Style GTM Checklist for dealershipAI

- [ ] Discovery: stakeholder interviews, role maps (GM, Marketing, Used, Service)
- [ ] Positioning: hero, RTB bullets, proof (evidence packets), pricing page narrative
- [ ] Design System: tokens, components, motion/audio, accessibility
- [ ] Activation: landing hero, explainer for Relevance Overlay, in-product playbooks, demo data
- [ ] KPIs: trial activations, schema fixes, AI citation share lift, leads/RO
- [ ] Feedback loop: in-app surveys, session replays, cohort funnels
