   $ ./quick-start.sh

2. Or manually:
   $ pip install -r requirements.txt
   $ cp .env.example .env
   $ export JWT_SECRET="$(openssl rand -base64 32)"
   $ uvicorn api.main:app --reload

3. Visit the documentation:
   → http://localhost:8000/docs (Swagger UI)
   → http://localhost:8000/redoc (ReDoc)
