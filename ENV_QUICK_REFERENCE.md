## ðŸš€ Quick Setup Commands

```bash

./scripts/setup-vercel-env.sh

# Add database URL

# Add Supabase keys
vercel env add NEXT_PUBLIC_SUPABASE_URL production

# Add OpenAI key

# Redeploy
vercel --prod
```
