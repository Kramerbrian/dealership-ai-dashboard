## Testing Your Copy

**The 3-Question Test**:

1. **Could this be said by a corporate robot?**
   - If yes ‚Üí rewrite

2. **Would a 10-year tower veteran actually say this?**
   - If no ‚Üí rewrite

3. **Does it make you smirk, even slightly?**
   - If no ‚Üí add personality

**The Sales Manager Test**:
- Read it out loud in your best sales manager voice
- If it sounds fake, it is fake
- If it sounds like a pep talk from corporate, burn it

## Key Principles

1. **Respect Intelligence**: Users are smart. Don't patronize.

2. **Acknowledge Reality**: Dealers are busy, skeptical, and have been burned before.

3. **Show, Don't Tell**: "You're losing $45K" beats "Our platform optimizes visibility"

4. **Self-Aware Humor**: We know we sound like another sales pitch. Address it.

5. **No False Urgency**: Use real deadlines (end of month, competitor activity)

6. **ROI Always**: Everything ties back to dollars and deals

7. **Dealership Context**: Reference real scenarios they live every day

## Voice Checklist

Before publishing any copy, ask:

- [ ] Would a real sales manager say this?
- [ ] Is it direct and honest?
- [ ] Does it reference actual dealership life?
- [ ] Is the humor dry, not cheesy?
- [ ] Does it respect the reader's intelligence?
- [ ] Is there a clear, practical next step?
- [ ] Would I roll my eyes if I read this?
- [ ] Does it sound like everyone else, or like us?
